SNES Advance v0.1
-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-

(Default controls)

Y   =   B
X   =   L
B   =   A
A   =   R
L   =   Select+L
R   =   Select+R

Select+Start+B+A brings up the emulator control menu.
Push Start to exit the menu.
Use Select+Up/Down to change screen offset.
